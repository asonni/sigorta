# Sigorta
sigorta admin dashboard

## Requirements
Node >= 8, </br>
npm >= 5

## Install The Dependencies
now type this line on your console
```
cd sigorta/server && npm install
```
```
cd sigorta/client && npm install
```

## Start Server
now type this line on your console

For the development mode
```
  cd sigorta/client && npm run dev
```
